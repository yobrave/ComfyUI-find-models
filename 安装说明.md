# ComfyUI Find Models 安装说明

## 安装步骤

1. **复制文件夹**
   - 将 `comfyUI-find-models` 文件夹复制到 ComfyUI 的 `custom_nodes` 目录
   - 路径示例：`ComfyUI/custom_nodes/comfyUI-find-models/`

2. **重启 ComfyUI**
   - 关闭 ComfyUI
   - 重新启动 ComfyUI

3. **使用**
   - 在 ComfyUI 界面中加载工作流
   - 点击工具栏上的 **"🔍 查找模型"** 按钮
   - 查看模型状态

## 目录结构

```
comfyUI-find-models/
├── __init__.py          # 节点注册
├── nodes.py             # 节点实现
├── server.py            # 服务器扩展（API端点）
├── web/
│   └── find_models.js   # 前端界面
├── README.md            # 说明文档
└── requirements.txt      # 依赖（通常不需要）
```

## 功能说明

### 按钮功能
- 点击 **"🔍 查找模型"** 按钮后，会自动分析当前加载的工作流
- 显示所有需要的模型及其状态

### 显示内容
1. **统计信息**
   - 总模型数
   - 已安装数量
   - 缺失数量

2. **按派系分类**
   - SDXL、Pony、Wan、Flux 等
   - 每个派系下的模型列表
   - 显示每个模型的安装状态

3. **缺失模型列表**
   - 列出所有缺失的模型
   - 显示模型类型和派系

## 注意事项

- 需要先加载工作流才能分析
- 模型检测基于文件名匹配
- 确保 ComfyUI 的模型目录配置正确

## 故障排除

如果按钮不显示：
1. 检查浏览器控制台是否有错误
2. 确认 `web/find_models.js` 文件存在
3. 重启 ComfyUI

如果API调用失败：
1. 检查 ComfyUI 控制台是否有错误
2. 确认工作流已正确加载
3. 检查 `server.py` 是否正确注册
